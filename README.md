## Introduction

A simple app to track products created with Node.js, Express and React.

## Startup

To run this project, Clone it and install each part locally and run them using npm:

```
$ cd ./server
$ npm i
$ npm run dev
```

Open a new terminal

```
$ cd ./client
$ npm i
$ npm run dev
```
